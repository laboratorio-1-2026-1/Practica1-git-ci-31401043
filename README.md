# Práctica de Control de Versiones con Git

Este repositorio tiene como propósito documentar y aplicar los conceptos fundamentales de **Git**. El objetivo principal es afianzar el flujo de trabajo estándar en el desarrollo de software, cubriendo desde la gestión de estados de archivos hasta la resolución de conflictos.

### Objetivos de la práctica:
* Configuración inicial y manejo del área de preparación (*Staging*).
* Gestión de ramas (*Branching*) y fusión de cambios (*Merging*).
* Uso de comandos esenciales: `commit`, `push`, `pull`.
